# labyrinthserver

In project directory run:

* To compile the project run `cmake -S . -B cmake-build`

* To build the project run `cmake --build cmake-build --target labyrinth_server -- -j 2`
                            
* To run the server `./cmake-build/labyrinth_server`

## Server response ##
```
pid: 'id'
tick timer started
bind done
server listening:
```
