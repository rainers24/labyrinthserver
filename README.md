# labyrinthserver

In project directory run command `cmake -S . -B build-direc` to compile a project.

To build the project run `cmake --build build-direc --target labyrinth_server -- -j 2`

To run the server `./build-direc/labyrinth_server`

## Server response ##
```
pid: 'id'
tick timer started
bind done
server listening:
```
